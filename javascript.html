  1. object - lấy ra các key là các name1, name2,...

  2. Chỉ lặp với dữ liệu mảng

  2. expression - không có tên và được gán vào biến - gọi sau khi định nghĩa

  1. function (các tham số) {

    }

    }

String

name.toString - chuyển sang string

name.length - tính chiều dài

name.indexOf('giá trị cần tìm', index)

  1. tính ra vị trí của giá trị cần tìm từ index
  2. không tìm được giá trị là -1

name.lastIndexOf('giá trị cần tìm') - tính ra vị trí cuối cùng

name.search('giá trị cần tìm')

name.slice(a, b)

  1. tạo ra biến mới từ vị trí a đến trước b
  2. giá trị biến name không thay đổi
  3. không có b thì cắt từ a đến hết chuỗi
  4. a, b là số âm khi tính ngược từ phải sang trái, a < b

name.replace('giá trị trước thay', 'giá trị sau thay') - thay cho giá trị đầu tiên

name.replace(/giá trị trước thay/g, 'giá trị sau thay') - thay toàn bộ

name.toUpperCase() - in hoa

name.toLowerCase() - chữ thường

name.trim() - bỏ khoảng trắng

name.split('X')

  1. cắt X khỏi name và trả về mảng
  2. X là điểm chung trong chuỗi

name.charAt(vị trí) - lấy kí tự theo vị trí

name[vị trí] - lấy kí tự theo vị trí

Number

Number(a) - chuyển string sang number

name.toFixed(x) - làm tròn số, lấy x số thập phân và chuyển sang string

number.toLocaleString('vn',{maximumFractionDigits: 2, minimumFractionDigits: 2}) - format số

parseInt(a) - lấy số nguyên

parseFloat(a) - lấy số thực

Math.round(a) - làm tròn

Math.abs(a) - giá trị tuyệt đối

Math.floor(a) - làm tròn xuống

Math.ceil(a) - làm tròn lên

Math.random() - ra số ngẫu nhiên dạng 0,.......

Math.min(a, b, c) - tìm số nhỏ nhất

Math.max(a, b, c) - tìm số lớn nhất

Array

Array.isArray(name) - kiểm tra kiểu dữ liệu Array

name.join('a') - a là ngăn cách các phần tử array khi chuyển sang chuỗi

name.push(giá trị cần thêm) - thêm giá trị vào cuối mảng

name.pop() - xoá giá trị ở cuối mảng

name.unshift(giá trị cần thêm) - thêm giá trị vào đầu mảng

name.shift() - xoá giá trị ở đầu mảng

name.slice(a, b)

  1. tạo ra mảng mới từ vị trí a đến trước b
  2. biến name không thay đổi
  3. không có b thì cắt từ a đến hết chuỗi
  4. a, b là số âm khi tính ngược từ phải sang trái

name.splice(a,b,c)

  1. a: vị trí bắt đầu thay đổi
  2. b: số giá trị muốn xoá từ a
  3. c: giá trị thêm vào từ a
  4. nếu chỉ có a,b xoá b phần tử từ vị trí a
  5. nếu chỉ có a xoá hết phần tử từ vị trí a

array1.concat(array2) - nối mảng

name.soft() - sắp xếp chữ cái từ a-z

name.reverse() - sắp xếp chữ cái từ z-a

name.soft(function(a,b){ return a-b}) - sắp xếp số theo thứ tự tăng dần

name.soft(function(a,b){ return b-a}) - sắp xếp số theo thứ tự giảm dần
