array - giá trị là [....]

  1. lấy giá trị 1 phần tử mảng - name[index]

object - giá trị là {name1:giá trị, name2: giá trị,.... }

  1. lấy key - Object.keys(name) - trả về mảng
  2. lấy giá trị đối tượng

    • name['name1']
    • Object.values(name) - trả về mảng

Vòng lặp

for(đk1; đk2; kđk3){}

  1. đk1: var i=0 chạy 1 lần đầu tiên
  2. đk2: điều kiện đúng
  3. đk3: i++

break và continue

  1. break - dừng vòng lặp với điều kiện nào đó
  2. continue - chạy tiếp với điều kiện

for lồng for

for in - for (var key in name){ code }

  1. object - lấy ra các key là các name1, name2,...
    • lấy giá trị của các key trong object - name[key]
  2. array - lấy ra các key là các số index
    • lấy giá trị của các key trong array - name[key]

for of - for (var value in name){ code }

  1. lấy ra các giá trị của mảng
  2. Chỉ lặp với dữ liệu mảng

while - giống for 100% chỉ khác cách viết

  1. var i =0
    while (i < 10){
    i++
    code
    }

do while

  1. var i =0
    do{
    i++
    code
    } while(i < 10)

Công dụng

  1. Người khác đọc code biết được chức năng hàm đó làm gì

Khai báo - function name(....){...}

  1. Gọi hàm name()
  2. đặt trong() là tham số, truyền vào giá trị khi gọi hàm

Các loại hàm

  1. declaration - function name(....){...} - gọi được trước khi định nghĩa
  2. expression - không có tên và được gán vào biến - gọi sau khi định nghĩa

Hàm không có giá trị trả về:

  1. function (các tham số) {
    Nội dung thực thi khi gọi hàm
    }

Hàm có giá trị trả về:

    }
