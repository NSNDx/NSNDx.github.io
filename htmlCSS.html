Thẻ form

form (block) - Có các thẻ con:

  1. label (inline) - Thuộc tính:
    1. for="id" để liên kết với thẻ input có tên id là "id"
  2. input (inline-block) - Thuộc tính:
    1. type=""
      • checkbox: ô tích nhiều
      • radio: ô cho chọn 1 khi đặt cùng thuộc tính name
      • text: nhập chữ
      • number: nhập số
      • password: nhập mật khẩu
      • email: nhập email
      • tel: nhập số điện thoại kèm pattern="[0-9]{3}-[0-9]{6}" định dạng số đt
      • date: nhập ngày tháng
      • file: nút để up file
      • color: chọn màu
      • range: thanh kéo trượt
      • reset: nút xoá trắng form
      • submit: nút gửi form chỉ có 1 cái sẽ load lại trang
      • button: nút gửi form không load lại trang
    2. placeholder="" nội dung chữ chìm hiện lên
    3. required bắt buộc điền vào ô input
    4. readonly, disabled chỉ đọc thông tin ô input và không thể chỉnh sửa
    5. autofocus trỏ chuột luôn nằm trong ô input
    6. maxlength="", minlength="" giới hạn kí tự gõ
    7. min="", max"" dùng cho nút chọn, có thể là số, ngày tháng,..
  3. textarea (inline-block) - Thuộc tính:
    1. resize="none" cố định kích thước
    2. textarea::placeholder css chỉnh thuộc tính placeholder
  4. select (inline) -> option (optgroup --> option) - Thuộc tính:
    1. multiple để chọn nhiều lựa chọn
    2. zise="" để chọn số lượng hiện thị
    3. selected của option để hiển thị đầu tiên của danh sách
  5. fieldset (block) -> legend nội dung - Gộp các nhóm input
  6. button (inline-block) - kèm theo thuộc tính type="button, submit hoặc reset", mặc định type = "submit"
  7. datalist chỉ định danh sách các tùy chọn được xác định trước cho phần tử input - ít dùng
